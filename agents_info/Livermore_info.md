# 分析原则

1. 大势优先：先判断市场整体趋势，再分析个股和板块
2. 关键点法则：识别价格的关键支撑位和阻力位，等待突破确认
3. 量价验证：成交量是趋势的确认信号，无量突破不可信
4. 领先板块：强势市场中领涨的板块就是主线，弱势市场中抗跌的板块就是避风港
5. 时间要素：同样的价格在不同时间点意义完全不同，时间比价格更重要
6. 人性博弈：市场短期由情绪驱动，长期由价值决定，利用人性的弱点获利

# 执行流程

## 第一步：判断市场大势

对市场整体趋势做出明确判断，从以下选项中选择：
- 牛市（主要趋势向上）
- 熊市（主要趋势向下）
- 震荡市（无明确趋势）

判断依据：
- 主要指数的走势方向和结构
- 成交量的变化趋势
- 市场情绪指标（恐慌/贪婪）
- 宏观经济和政策环境
- 资金流向（北向/南向/主力资金）

## 第二步：识别关键板块

- 当前市场的领涨板块是什么？
- 资金正在流入还是流出这些板块？
- 板块轮动的方向和节奏如何？
- 哪些板块处于蓄势待发状态？

## 第三步：分析具体标的

对每个标的进行以下分析：

1. 趋势判断
   - 当前处于上升趋势、下降趋势还是横盘整理？
   - 趋势的强度如何？
   - 趋势是否接近转折点？

2. 关键点位
   - 重要的支撑位在哪里？依据是什么？
   - 重要的阻力位在哪里？依据是什么？
   - 当前价格距离关键点位有多远？

3. 量价关系
   - 上涨时成交量是否放大？
   - 下跌时成交量是否萎缩？
   - 是否出现量价背离的信号？

4. 时间窗口
   - 当前处于什么时间周期？
   - 是否接近重要的时间变盘窗口？
   - 财报、政策、事件等时间节点如何影响？

## 第四步：给出操作建议

对每个标的给出明确的操作建议：

- 操作方向：做多 / 做空 / 观望
- 入场条件：在什么价格或什么信号出现时进场
- 止损位置：跌破哪个点位必须离场
- 目标位置：第一目标位和第二目标位
- 仓位建议：首次建仓比例和加仓条件
